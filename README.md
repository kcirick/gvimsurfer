gVimSurfer
==========
Web browser that looks and works like gVim

Description:
------------
  - Features/functions inspired by jumanji and vimprobable
  - organized in a way it makes sense
  - Requires following as dependencies:
   - gtk+-2
   - webkit-1.0
   - libsoup
